# 🎉 BGEvents

BGEvents е уеб базирана платформа (портална система) за управление и популяризиране на събития и събитийни услуги. Тя предоставя ефективен дигитален канал за комуникация между организатори на събития и крайни потребители.

---

## 📌 Основна цел

Целта на BGEvents е да обедини в една платформа:
- организатори на събития
- доставчици на услуги
- потребители, търсещи събития

Платформата улеснява:
- публикуването и управлението на събития
- търсенето и филтрирането на събития
- комуникацията между участниците

---

## 👥 Целеви потребители

- 🧑‍💼 Администратор
- 🎤 Организатор на събития
- 👤 Краен потребител (клиент)

## ⚙️ Основни функционалности

### 🔐 Регистрация и вход
- Създаване на акаунти
- Вход в системата
- Различни потребителски роли

### 🔎 Търсене и филтриране
- По тип събитие
- По дата
- По локация
- По категория

### 📅 Управление на събития
- Добавяне на събития
- Редактиране
- Изтриване

### 👤 Потребителски профили
- Управление на лични данни
- Управление на събития (за организатори)
- Административен контрол (за администратор)

### 💬 Комуникация
- Изпращане на запитвания към организатори

## 🏗️ Архитектура

Проектът използва многопластова архитектура:
